# Research / 研究方向

## Intelligent Software / 智能软件

面向工业互联网、工业控制系统和复杂装备场景，研究智能软件架构、软件定义控制、工业软件平台与智能化开发方法。

## Edge Computing / 边缘计算

围绕工业物联网和边缘智能系统，研究边缘计算架构、边缘数据采集、边缘设备协同、边缘容器与工业现场实时计算等问题。

## Embedded Systems / 嵌入式系统

研究嵌入式 PLC、工业控制嵌入式实时操作系统、可编程控制器平台软件、工业设备接入与协议解析等方向。

## Intelligent Control / 智能控制

围绕工业机器人、复杂装备、柔性制造和智能产线，研究智能控制、运动控制、视觉伺服控制、轨迹规划与优化等技术。

## Embodied Intelligent Robotics / 具身智能机器人

面向智能机器人与工业机器人应用，研究机器人感知、控制、协同、任务规划和具身智能软件系统。
